## üíª Local Development

If you'd like to explore the codebase or run it locally:

1.  **Clone & Install:**
    ```bash
    git clone https://github.com/rohanakode490/portfolio.git
    cd portfolio
    npm install
    ```

2.  **Environment Setup:**
    Configure your `.env.local` for the contact form functionality:
    ```env
    BREVO_API_KEY=...
    BREVO_USER_NAME=...
    BREVO_USER_EMAIL=...
    BREVO_TO_MAIL=...
    BREVO_TO_NAME=...
    ```

3.  **Run:**
    ```bash
    npm run dev
    ```
